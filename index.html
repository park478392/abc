프로젝트

팀 프로젝트 설명

< 주요 사이버 보안 위협과 예방법 안내 웹사이트 >

본 프로젝트는 일상생활에서 사용자가 쉽게 접할 수 있는 다양한 사이버 보안 위협을 소개하고, 이에 대한 예방법을 체계적으로 안내하는 웹사이트를 제작하는 것을 목표로 합니다. 최근 피싱, 비밀번호 해킹, 개인정보 유출 등 다양한 보안 문제가 증가하고 있지만, 일반 사용자들은 이러한 위험성을 정확히 인지하지 못하는 경우가 많습니다. 따라서 본 웹사이트에서는 단순한 개념 설명에 그치지 않고, 실제 사례와 비교를 통해 사용자가 보안의 중요성을 보다 직관적으로 이해할 수 있도록 구성합니다.

사이트는 보안 정보를 단순히 나열하는 형식이 아니라, 리스트, 표, 카드형 레이아웃, 미디어 요소 등을 활용하여 사용자가 쉽고 흥미롭게 내용을 확인할 수 있도록 구성합니다. 주요 콘텐츠는 다음과 같이 구성됩니다. 메인 홈 화면 사이트 제목과 소개 문구를 배치하고, 대표 보안 수칙을 리스트 형식으로 제시합니다. 보안 이슈 페이지 피싱, 랜섬웨어, 개인정보 유출 등 주요 보안 이슈를 다룹니다. 또한 관련 뉴스 이미지나 미디어 자료를 함께 넣어 실제 사례와 연결해 보여줍니다.

OWASP 페이지 OWASP Top 10의 주요 취약점을 소개하는 페이지로, 각 취약점에 대해 설명, 발생 원인, 대응 방법 등을 표 형식으로 정리합니다. 이를 통해 웹 보안의 핵심 개념을 보다 체계적으로 이해할 수 있도록 합니다. 보안 수칙 및 대응 가이드 페이지 강한 비밀번호 생성, 공용 와이파이 사용 시 주의사항, 의심스러운 링크 클릭 금지 등 실생활에서 바로 적용할 수 있는 보안 수칙을 안내합니다. 리스트를 통해 해야 할 행동과 하지 말아야 할 행동을 구분하고, 표를 통해 특정 상황에서의 위험한 행동과 안전한 행동을 비교하여 제시합니다.
