팀 프로젝트 개요

📌 프로젝트 개요

프로젝트명: 주요 사이버 보안 위협과 예방법 안내 웹사이트

목표: 일반 사용자들이 쉽게 이해할 수 있는 사이버 보안 정보 제공

배경: 최근 피싱, 비밀번호 해킹, 개인정보 유출 등 다양한 보안 문제가 증가하고 있음

필요성: 일반 사용자들은 이러한 위험성을 정확히 인지하지 못하는 경우가 많아 보안 교육이 필요

🎯 주요 목표

✓ 실제 사이버 위협 사례 소개

✓ 체계적인 예방법 안내

✓ 사용자 친화적 인터페이스 구성

✓ 보안의 중요성 인식 제고

✓ 실생활에서 바로 적용 가능한 정보 제공

GitHub

중간고사에서 기말고사로 발전한 점

기말고사 프로젝트는 단순한 보안 정보 소개를 넘어, 최신 이슈 확인, 취약점 실습, 위험 진단, 보안 퀴즈까지 직접 체험할 수 있는 교육형 웹사이트로 발전했습니다.

보안 이슈

최신 이슈와 영상 콘텐츠 확장

보안 이슈 페이지에서 최신 보안 뉴스를 카드로 정리하고, 피싱·랜섬웨어·취약점 등 카테고리 필터를 추가했습니다. 또한 유튜브 보안 영상을 자동으로 불러오고 채널별로 분류해 학습 자료를 더 쉽게 찾을 수 있게 했습니다.

보안 실습

BAD / GOOD 비교 실습 구현

보안 실습 페이지에서는 XSS, 접근 통제, 무차별 대입 공격을 취약한 환경과 안전한 환경으로 나누어 비교할 수 있게 했습니다. 공격 결과와 방어 결과를 직접 확인하고, 핵심 JavaScript 코드까지 비교할 수 있도록 구성했습니다.

위험 진단

개인 보안 상태 점검 기능

위험 진단 페이지에서 비밀번호 안전도, 피싱 메일 위험도, URL 안전도 검사를 제공했습니다. 메인 페이지에는 보안 습관 체크리스트와 최신 기사 기반 위험 점수도 추가해 사용자가 자신의 보안 상태를 더 구체적으로 확인할 수 있게 했습니다.

보안 수칙

탭형 보안 수칙과 퀴즈 추가

보안 수칙 페이지는 계정 관리, 기기 보안, 피싱 대응, 데이터 보호 탭에 따라 카드 내용이 동적으로 바뀌도록 개선했습니다. O/X 보안 상식 퀴즈와 결과 화면을 추가해 사용자가 자신의 보안 이해도를 점검할 수 있게 했습니다.
