팀 프로젝트 개요

📌 프로젝트 개요

프로젝트명: 주요 사이버 보안 위협과 예방법 안내 웹사이트

목표: 일반 사용자들이 쉽게 이해할 수 있는 사이버 보안 정보 제공

배경: 최근 피싱, 비밀번호 해킹, 개인정보 유출 등 다양한 보안 문제가 증가하고 있음

필요성: 일반 사용자들은 이러한 위험성을 정확히 인지하지 못하는 경우가 많아 보안 교육이 필요

🎯 주요 목표

✓ 실제 사이버 위협 사례 소개

✓ 체계적인 예방법 안내

✓ 사용자 친화적 인터페이스 구성

✓ 보안의 중요성 인식 제고

✓ 실생활에서 바로 적용 가능한 정보 제공

주요 콘텐츠 구성

1️⃣ 메인 홈 화면

사이트의 첫 인상을 담당하는 홈 페이지

  • 사이트 제목과 소개 문구
  • 대표 보안 수칙을 리스트 형식으로 제시
  • 사용자를 안내하는 네비게이션

2️⃣ 보안 이슈 페이지

실제 발생하는 주요 보안 위협 소개

  • 피싱(Phishing) 공격
  • 랜섬웨어(Ransomware)
  • 개인정보 유출 사례
  • 관련 뉴스 이미지와 미디어 자료 포함

3️⃣ OWASP Top 10

웹 보안의 핵심 취약점 안내

  • SQL Injection
  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • 각 취약점에 대한 설명, 원인, 대응 방법

4️⃣ 보안 수칙 및 가이드

실생활에서 바로 적용 가능한 조언

  • 강한 비밀번호 생성 방법
  • 공용 WiFi 사용 시 주의사항
  • 의심스러운 링크 클릭 금지
  • 위험한 행동 vs 안전한 행동 비교

프로젝트 세부 사항

기술 스택

프론트엔드:

  • HTML5 - 의미있는 마크업
  • CSS3 - 반응형 디자인
  • JavaScript - 동적 기능

디자인: 다크모드, 카드형 레이아웃, 미디어 요소

개발 프로세스
