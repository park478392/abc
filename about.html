웹 개발을 배우고 있는 학생

요즘 배우고 있는 것

HTML로 페이지 구조를 잡고, CSS로 화면을 정리하고, JavaScript로 버튼 클릭, 화면 변화, 카드 인터랙션 같은 기능을 구현해 보았습니다. 팀 프로젝트에서도 제가 맡은 JavaScript 기능을 완성하면서 실제 페이지에 적용하는 경험을 했습니다.

관심 있는 부분

단순히 예쁘게 보이는 화면보다, 메뉴를 눌렀을 때 자연스럽게 이동하고 내용을 쉽게 찾을 수 있는 페이지에 관심이 있습니다. 팀 프로젝트를 하면서는 제가 맡은 부분을 책임지고 완성하는 과정도 배우고 있습니다.

지금 할 수 있는 것

  • HTML 태그를 사용해 기본적인 페이지 구조 만들기
  • CSS로 색상, 여백, 카드, 반응형 레이아웃 조정하기
  • JavaScript로 버튼 클릭, 정보 표시, 퀴즈 결과 출력 구현하기
  • 수업 과제 내용을 정리해서 웹 페이지로 보여주기

앞으로 더 익히고 싶은 것

  • JavaScript 코드를 더 깔끔하게 정리하고 재사용하기
  • 페이지 구조를 더 깔끔하게 나누고 유지보수하기
  • 사용자가 보기 편한 화면 구성과 반응형 디자인 연습하기
  • 팀 프로젝트에서 구현한 기능을 더 발전시켜보기
